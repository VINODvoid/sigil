# Sigil — Naming Context & Decisions

---

## Why "Sigil"

**Definition:** A sigil is a symbol or seal believed to have magical power — historically used to bind, authenticate, or invoke.

**Why it fits:**
- **Cryptographic seal metaphor** — a Sigil binds an agent to a principal
- **Short, memorable** — 2 syllables, 5 letters
- **Pronounceable** — /ˈsɪdʒəl/ (SIJ-uhl)
- **Works as noun and verb** — "needs a sigil" / "sigil your agent"
- **No heavy prior associations** in crypto (not a token, not a brand)
- **Distinctive without being weird** — fits infrastructure protocol tone
- **Mystical but technical** — matches the Solana ecosystem aesthetic

## Product Naming Conventions

**Core protocol:** Sigil
**Individual credentials:** "a Sigil" (lowercase when talking about instances)

**Actions/verbs:**
- To "sigil" an agent = to issue a credential
- To "verify" a Sigil = check authenticity
- To "revoke" a Sigil = cancel it
- To "attest" to a Sigil = third-party signature

**Packages:**
- `@sigil/sdk` — core SDK
- `@sigil/x402` — x402 middleware

## Usage in Copy

### Do
- "Every agent carries a Sigil."
- "Verify the Sigil before accepting payment."
- "Issue a Sigil to your agent in one click."
- "Without a Sigil, agents can't transact."

### Don't
- Capitalize inconsistently ("sigil" vs "Sigil" — pick one style for instances)
- Use jargon without explaining (first mention: "Sigil (a cryptographic agent credential)")
- Over-mystify — it's infrastructure, not magic (mystical metaphor is subtle)

## Alternate Name Deep Dive (If Needed)

If Sigil doesn't work, here's the analysis framework:

### Name must pass 4 tests:
1. **Pronunciation test** — say it 3 times. Any stumble = reject.
2. **Pitch test** — can you pitch in 10 words using this name?
3. **Availability test** — domain + handle + no crypto collision
4. **Story test** — does the name have a 1-sentence origin story?

### Top Alternates Re-evaluated

**Aegis** — strong meaning (shield), short, easy to say. Risk: maybe too "security tool" vibes.

**Janus** — perfect metaphor (gates/duality), memorable. Risk: pronunciation (JAY-nus vs YAH-nus), some might not know the god.

**Vouch** — simple, verb, intuitive. Risk: too common English word, might lack gravitas.

**Fides** — Latin trust goddess, short. Risk: pronunciation (FEE-days vs FYE-deez), less recognized.
